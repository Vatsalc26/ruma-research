HTTPX app shortcut before removal.

Curated benchmark note from the official releases page. In HTTPX 0.27.0, the `app=...` shortcut was still present but deprecated. The release notes told users to prefer the explicit transport style with `transport=httpx.WSGITransport()` or `transport=httpx.ASGITransport()` instead.
