FastAPI Python support after the later drop.

Curated benchmark note from the official release-note stream. FastAPI 0.129.0 dropped support for Python 3.9, so Python 3.10+ remained supported. The official FastAPI release notes updated internal types for Python 3.10 and migrated docs from Python 3.9 to Python 3.10.
