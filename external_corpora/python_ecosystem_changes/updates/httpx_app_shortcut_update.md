HTTPX app shortcut after removal.

Curated benchmark note from the official releases page. In HTTPX 0.28.0, the `app` argument has now been removed. The explicit transport style with `transport=httpx.WSGITransport()` or `transport=httpx.ASGITransport()` remained the documented path after the removal.
