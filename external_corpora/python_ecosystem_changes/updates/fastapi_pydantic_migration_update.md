FastAPI Pydantic migration after the cleanup started.

Curated benchmark note from the official release notes. FastAPI later dropped support for Pydantic v1, first keeping only short temporary support for Pydantic v2's pydantic.v1 and then dropping pydantic.v1 support outright. Pydantic v2 still remains the supported migration target, and FastAPI later ran performance tests only on Pydantic v2.
