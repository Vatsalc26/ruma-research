# RUMA Agent Manual v2

RUMA Agent remains the command surface for the current memory-backed assistant workflow.
The default retrieval top_k in RUMA Agent is now `4`.
This update supersedes the earlier top_k setting for the standard agent path.
The answer mode in RUMA Agent remains `citation-first`.
Citation-first mode still means the system should prefer grounded answers with source references instead of pretending to know more than it can support.
This manual captures the updated v2 retrieval default while preserving the answer mode.
