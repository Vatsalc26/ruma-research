# Helios Packager Manual v2

Helios Packager is still the release packaging tool used in the delivery path.
The default publish target is now `staging_registry`.
This change supersedes the earlier `internal` publish target for normal release preparation.
The main Helios Packager manifest file remains `helios.yaml`.
The manifest file still lives in the repository root for each package.
This manual captures the updated v2 publish target while preserving the manifest convention.
