# Release Notes v0.2.1

`v0.2.1` is a packaging and metadata refinement release for the frozen `RUMA V2` software artifact.

The scientific claim of `RUMA V2` is unchanged. This version improves the public release boundary and finalizes the canonical DOI metadata associated with the `V2` package.

## Canonical DOI Records

- software DOI: `10.5281/zenodo.19655779`
- preprint DOI: `10.5281/zenodo.19652555`

## Key Updates

- refined the public software package to keep the release centered on the research artifact itself
- removed support files that were not necessary to the public software snapshot
- clarified the historical relationship between `V1` and `V2` in the public README
- finalized public-facing DOI metadata for the software release and associated preprint

## Release Framing

`RUMA V2` should be read as an interleaved routed updatable memory architecture family for:

- explicit updates
- retained guidance
- stale-memory suppression
- grounded evidence control

## Claim Boundary

This release does **not** claim:

- a broad standalone foundation model
- a permanently settled backbone comparison
- a general solution to hallucination
- a universal replacement for standard language-model backbones
