# Release Notes v0.2.1

`v0.2.1` is a corrective public software release for the frozen `RUMA V2` package.

This version does **not** widen the scientific claim of `V2`. It exists to clean the public artifact boundary and separate the public research package from internal release-ops files.

## What Changed Relative To v0.2.0

- removed release-ops/support files that did not need to be part of the public software artifact
- tightened the public package boundary around the actual research release
- clarified `V1` versus `V2` history in the public README
- updated public release metadata from `v0.2.0` to `v0.2.1`
- kept the scientific framing and benchmark story of `RUMA V2` unchanged

## What This Release Still Is

`RUMA V2` remains an interleaved routed updatable memory architecture family for:

- explicit updates
- retained guidance
- stale-memory suppression
- grounded evidence control

## What This Release Still Does Not Claim

- a broad standalone foundation model
- a permanently settled backbone comparison
- a general solution to hallucination
- a universal replacement for standard language-model backbones

## Practical Reason For This Version

The first public `V2` software release was slightly broader than necessary and exposed some internal release-ops files. `v0.2.1` corrects that packaging boundary so the public repo reads more like a deliberate research artifact and less like a workshop snapshot.
