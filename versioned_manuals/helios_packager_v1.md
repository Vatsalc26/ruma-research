# Helios Packager Manual v1

Helios Packager is the release packaging tool used in the internal delivery path.
The default publish target is `internal`.
Release operators should confirm the publish target before starting a packaging run.
The main Helios Packager manifest file is `helios.yaml`.
The manifest file lives in the repository root for each package.
This manual captures the stable v1 publish target and manifest convention.
