# RUMA Agent Manual v1

RUMA Agent is the command surface for the current memory-backed assistant workflow.
The default retrieval top_k in RUMA Agent is `2`.
The answer mode in RUMA Agent is `citation-first`.
Citation-first mode means the system should prefer grounded answers with source references instead of pretending to know more than it can support.
This manual records the stable v1 retrieval and answer defaults for the agent shell.
