# Tidal Recovery Runbook

For disaster-recovery restores, the default snapshot mode in Tidal Sync guidance is `full_mirror`.
This runbook covers recovery-only behavior for a narrower emergency context.
It does not supersede the main sync manual for routine replication cycles.
Instead, it provides an active recovery-specific recommendation for restore-heavy workflows.
Operators should compare this runbook with the main sync manual when deciding whether disaster-recovery mode applies.
