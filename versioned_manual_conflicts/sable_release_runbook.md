# Sable Release Runbook

For regulated release windows, the default rollout mode in Sable Deploy guidance is `serial_guard`.
This runbook covers regulated release handling for a narrower operating context.
It does not supersede the main deployment manual for routine staged releases.
Instead, it provides an active release-specific recommendation for stricter rollout control.
Operators should compare this runbook with the main deploy manual when deciding whether regulated release mode applies.
