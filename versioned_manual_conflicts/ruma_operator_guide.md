# RUMA Operator Guide

For large retrieval jobs, the default retrieval top_k in RUMA guidance is `6`.
The operator guide covers large-job handling for high-load retrieval sessions.
This guide does not supersede the main deployment or product manual.
Instead, it provides an active operator-specific recommendation for a narrower workload context.
Operators should compare this guide with the main agent manual when deciding whether the large-job mode applies.
