# Atlas Incident Runbook

For live incident escalations, the default alert mode in Atlas Monitor guidance is `always_page`.
This runbook covers incident-only escalation behavior for narrow emergency conditions.
It does not supersede the main product manual for routine service watches.
Instead, it provides an active incident-specific recommendation for a different operating context.
Operators should compare this runbook with the main monitor manual when deciding whether incident escalation mode applies.
